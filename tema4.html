Emergency Site

This project was all about redesigning and developing a website using HTML, CSS, JavaScript, and custom graphics. It gave me the chance to be more creative with both the visual design and the interactive elements while building on my front-end development skills.

From Design to Development

Throughout the project, I worked with visual concepts, UI elements, SVG graphics, and interactive features. This project gave me the chance to combine creativity and functionality while creating a website with its own visual style. It also helped me understand how design and interactive elements can work together to make a website more engaging and easier to use.

Reflection

This project helped me build on the skills I had learned in earlier themes while trying out new tools and techniques. Working with JavaScript and interactive elements gave me a better understanding of how design and functionality come together to create a more engaging experience for users.
